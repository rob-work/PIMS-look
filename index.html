PIMS now exposes its most signifcant data through a RESTful API using the HTTP protocol for read (GET), write (POST) and update (PUT). Data read from PIMS is returned in XML format by default, but can also be retrieved in JSON format by appending "=json" to the end of each API call. Write and update functions require data to be passed in XML format only (there is no JSON option).
